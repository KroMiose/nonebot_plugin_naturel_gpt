## 🧑‍💻 开发指南

> 通过以下几步操作即可快速开始开发本插件

1. 克隆本仓库

```bash
git clone https://github.com/KroMiose/nonebot_plugin_naturel_gpt.git
```

2. 进入目录并安装依赖

```bash
cd nonebot_plugin_naturel_gpt
pip install poetry  # 需要提前安装 Python 环境: 推荐 Python 3.10
poetry install --dev
```

3. 运行 Bot

```bash
nb run
```
